## Image Guidelines

- **Format:** JPG for photos, PNG for diagrams/screenshots with transparency
- **Optimization:** Compress all images to <200KB each using tools like TinyPNG
- **Resolution:** 2x resolution for retina displays
- **Alt text:** Descriptive alt text is already provided in HTML
- **Naming:** Use lowercase, hyphens for spaces, descriptive names

## Temporary Solution

Until you add real images:
1. The site will show placeholder boxes with dashed borders
2. These clearly indicate where images should go
3. Everything else is fully functional

## Adding Your Images

1. Export images from Figma, Fusion 360, or take screenshots
2. Optimize them using TinyPNG.com
3. Name them according to the list above
4. Place them in this directory
5. The site will automatically display them (no code changes needed)
